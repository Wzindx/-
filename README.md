# UniversalImageStudio

UniversalImageStudio 是一个面向 Android 的通用图像生成工坊，支持文生图、图生图、兼容接口调用、后台任务处理、历史记录和首次配置引导。

当前主线版本为 **v1.8**，应用包名为：

```text
com.yang.emperor
```

> 旧版 `v1.7` 已下架，避免用户下载到旧包名安装包。

## 核心能力

### 图像创作

- **文生图**：输入提示词后直接生成图像。
- **图生图**：选择参考图后，基于图片和提示词生成新图像。
- **统一按钮文案**：无论文生图还是图生图，首页按钮统一显示为“生成图像”，不暴露队列、编辑等技术词。
- **结果预览**：生成完成后可在结果区域查看图像。
- **手动下载**：生成结果默认不自动保存，由用户按需下载。

### 接口与模型

- 支持配置接口地址、API Key 和模型名称。
- 支持首页快速调整模型与生成参数。
- 支持多种接口模式，适配不同服务端实现：
  - Images API
  - Responses API
  - Generations 兼容模式
- 模型、尺寸、画质、输出格式、背景、生成数量等参数可按需调整。

### 后台任务

- 点击“生成图像”后，任务会进入后台处理。
- 后台任务使用 Activity 级协程作用域，不再绑定 Compose 页面生命周期。
- 切换到历史页、设置页或弹窗关闭时，生成任务不会因为页面离开 composition 而中断。
- 修复兼容接口文生图中可能出现的：

```text
The coroutine scope left the composition
```

### 历史记录

- 历史页展示任务状态：
  - 处理中
  - 成功
  - 失败
- 失败任务会保留错误信息，便于排查接口或参数问题。
- 生成成功后可从结果预览继续查看或下载。

### 首次引导与设置

- 首次打开应用时，如果接口信息未配置，会进入引导页。
- 设置页保留“再来一次引导”，方便重新配置接口。
- 首页不再常驻显示配置提示，减少干扰。
- 设置页集中管理连接配置，首页聚焦创作流程。

### 首页与交互

- 首页创作区支持滚动，接口与模型、生成参数等区域可自然滑出屏幕。
- 底部导航视觉已弱化大色块，减少压迫感。
- 生成按钮优化了反色和禁用态对比。
- 生成参数、接口与模型等底部面板改为半屏弹窗：
  - 不再占满整个屏幕
  - 内容内部可滚动
  - 支持下滑关闭
  - 点击空白区域也可关闭

## 使用流程

1. 安装 Release 页面中的 `app-release.apk`。
2. 首次打开时，根据引导填写：
   - Base URL
   - API Key
   - 模型名称
3. 回到首页输入提示词。
4. 可选：选择参考图。
5. 点击“生成图像”。
6. 切到历史页查看后台任务状态。
7. 生成成功后在结果预览中查看或下载图片。

## 安装说明

请从 GitHub Release 下载最新版本：

```text
v1.8 / app-release.apk
```

当前包名：

```text
com.yang.emperor
```

如果手机上已安装旧包名版本，它会被系统视为另一个应用。建议手动卸载旧版后再安装新版。

## 开发与构建

项目为 Android Kotlin / Jetpack Compose 应用。

GitHub Actions 会自动构建 Release APK。  
本地环境具备 Gradle 时可执行：

```bash
gradle :app:assembleRelease
```

主要文件：

```text
app/src/main/java/com/yang/emperor/MainActivity.kt
.github/workflows/release.yml
README.md
```
